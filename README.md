## Предусловия

- Установленный **Python 3.8+**
- Установленный **pip**
- Установленный **git**

## Установка

1. Клонировать репозиторий:

   ```bash
   git clone https://github.com/KRblL/QA-internship-avito.git
   ```

2. Перейти в директорию проекта:

   ```bash
   cd QA-internship-avito
   ```

3. Создать виртуальное окружение:

   ```bash
   python -m venv venv
   ```
   ИЛИ
   ```bash
   python3 -m venv venv
   ```

4. Активировать виртуальное окружение:

   - **Windows**:

     ```bash
     venv\Scripts\activate
     ```

   - **Linux / macOS**:

     ```bash
     source venv/bin/activate
     ```

5. Установить зависимости:

   ```bash
   pip install -r requirements.txt
   ```

## Запуск тестов

Для запуска всех тестов выполнить:

```bash
pytest -v
```

Результаты непройденных тестов соответствуют найденным дефектам и описаны в файле **BUGS.md**.
